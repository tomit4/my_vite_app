# Vue 3 + Vite + Vue-Router + Vuex + Pug

A simple template for playing around Vite + Vue3. Much faster than the Vue-Cli!

## Installation:

`npm init vite@latest my-vue-app --template vue`

`cd my-vue-app`

`npm install`

`npm install pug`

`npm install vue-router@next`

`npm install vuex@next`

`npm install axios`

`npm run dev`
